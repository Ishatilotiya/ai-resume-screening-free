# AI Resume Screening System (Free Version)

## 📌 Overview
This project is an AI-based Resume Screening System that evaluates candidate resumes against a job description.

It performs:
- Skill extraction
- Resume-job matching
- Candidate scoring (0–100)
- Explanation generation

---

## 🚀 Features
✔ Extracts skills, tools, and experience from resume  
✔ Matches resume with job requirements  
✔ Calculates score based on keyword matching  
✔ Provides explanation for evaluation  

---

## 🧠 Pipeline Flow
Resume → Extraction → Matching → Scoring → Explanation

---

## 🛠 Tech Stack
- Python
- Hugging Face Transformers
- NLP (Text Generation)
